DJIA Daily Closing Prices: End User Document

Ramana Sonti

4/30/2017

Synopsis:

The shiny app DJIA Daily Closing Prices allows the user to select the year from a dropdown or a slider on the left panel to display the trend chart for that year on the main panel on the right.

Screenshot for DJIA Daily Closing Prices

User Interface:

The daily snapshots of DJIA at the close of the market for each year from 2001 and 2016 are in scope. The user should be able to take a look at the closing price of the index for any given day during this period.

The user is presendted with two input options (a slider and a dropdown) on the left panel to select the year he/she is interested in. Once the year is selected, a trend chart for the year selected will be displayed on the main panel on the right side. The interactive chart is drawn using plotly and the user can hover over the curve to get the closing price for any given market day during that year.

Right above the plot, a message that says Server getting data for the year xxxx is updated while the server is executing the api to fetch the data from finance.yahoo.com.

Internal Details:

The application uses shiny package to leverage the web framework it offers in a clien-server paradigm. The client side of the code creates user inreface components and sends the user inputs back to the server which in turn processes the data and sends the output back to the client for rendering.

The server component uses getSymbols api from quantmod package to fetch the data for a given year and renders the plot using plot_ly api from plotly package.
